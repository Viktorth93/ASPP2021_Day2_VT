simple_math.poly_first(x, a0, a1)¶

Returns the value of a first order polynomial with parameters a0 and a1 evaluated at x.

Parameters
xVariable value for the evaluation of the function.
a0Constant term.
a1Coefficient multiplying the linear term.

See also

poly_second

Examples

>>> simple_math.poly_first(1,2,3)
5
