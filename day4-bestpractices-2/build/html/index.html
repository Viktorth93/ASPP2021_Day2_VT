Welcome to SimpleMath’s documentation!¶

A collection of simple math operations

simple_math.poly_first(x, a0, a1)¶

Returns the value of a first order polynomial with parameters a0 and a1 evaluated at x.

simple_math.poly_second(x, a0, a1, a2)¶

Returns the value of a second order polynomial with parameters a0, a1, and a2 evaluated at x.

simple_math.simple_add(a, b)¶

Returns the sum of two numbers.

simple_math.simple_div(a, b)¶

Returns the quotient of two numbers.

simple_math.simple_mult(a, b)¶

Returns the product of two numbers.

simple_math.simple_sub(a, b)¶

Returns the difference between two numbers.
